# Unterwater Zoo Race

## Server

Game server is reachable under wss://ggj-2025.onrender.com

## Map Format

See [example.txt](example.txt).

Legend:

* `#` ... Wall
* `.` ... Movable water, drains oxygen
* `O` ... The (locked) exit
* `A` - `H` ... Exit open switches
* `o` ... Oxygen, restores oxygen
* `0` - `7` ... Player start positions
